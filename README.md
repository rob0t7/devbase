devbase cookbook
================

Install a development environment suitable for Rails or NodeJS development

Requirements
------------

### Cookbooks

The following cookbooks are direct dependencies:
    * rbenv
    * postgresql
    * database
    * nodejs
    * nginx

### Platforms

The following platforms are supported and tested:
    * CentOS 6.4
